# Radice-Christmas-Harmony


Web project developed for the Radice Christmas Harmony contest, implemented as a digital advent calendar. The site uses PHP and MySQL for user account management, institutional login authentication, likes system, password recovery, OTP generation and verification, and controlled access to daily unlocked content.



Progetto web sviluppato per il concorso Radice Christmas Harmony, realizzato come calendario dell’avvento digitale. Il sito utilizza PHP e MySQL per la gestione degli account utente, autenticazione con login istituzionale, sistema di like, recupero password, generazione e verifica OTP e controllo dell’accesso ai contenuti sbloccati giornalmente.
